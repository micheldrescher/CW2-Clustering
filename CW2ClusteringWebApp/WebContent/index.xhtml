Cloud landscape clustering

About the Cloud landscape clustering tool:

The CloudWATCH project, partially funded through the EC FP7 (Contract No. 610994), ran from September 2013 to October 2015. It developed a clustering methodology in order to gain insight in commonalities of European cloud computing projects across the entire Cloud computing landscape. The objective of this empirical analysis is to discover distinct groups of projects that are consistent in their relationship to a set of well-defined characteristics. The seed dataset for this analysis was compiled by CloudWATCH representing 38 European projects and scored against the full set of 13 NIST defining characteristics on an interval scale.

This online application builds upon this manual clustering work, offering an interactive extension to this initial manual clustering analysis. This web application provides the following functionality:

  1. View base data.
    Inspect the original scoring data as provided as part of the seed dataset, or subsequently added by other users and validated by CloudWATCH2 representatives.
  2. Cluster analysis
    Perform interactive cluster analysis in order to gain individual insight. Select one of the preset NIST cloud computing characteristics, or configure an individual set of characteristics. Set a maximum cluster distance to control the size and focus of computed clusters.
  3. Add your scores.
    Enter your individual activity or project's scores to have it included in the extended base dataset. Note: Before your scores will be included in any cluster analysis, CloudWATCH2 representatives will need to validate the data to prevent tainting the base data set and degradation of the analysis quality.
  4. Manage data.
    This functionality is restricted to named CloudWATCH representatives empowered to access the provided data, and validate the entered scores.
